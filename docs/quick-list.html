quick-list

quick-list produces abbreviated SMF files, for use in quick-look reductions.


USAGEquick-list
INPUTprogram prompts for name of SMF file, and information on which apertures to choose
OUTPUTfilename_short.SM the abbreviated SMF file
PARAMETERSnone


Details:

quick-list prompts for the name of the SMF file to be processed. A subset of the file can be produced either by picking every nth object, or by providing the names of specific objects to be included.
